Массивы и строки.

Задача 1. Определить сколько раз каждый элемент встречается в массиве.
Генерируется массив из случайных объектов (строк, чисел, массивов). Программа определяет, сколько объектов и какого типа встречается в массиве.

Задача 2. Создать массив который содержит только уникальные значения исходного массива.
Наличие пользовательской функции обязательно. Функция принимает на вход массив. После обработки массива, функция возвращает массив с уникальными элементами из исходного массива.

Задача 3. Создать функцию, которая группирует студентов по возрасту.
Имеется массив, где каждый чётный элемент это имя студента, каждый нечетный - возраст.
На фход функции передаётся массив с информацией про студента. Функция печатает в столбик информацию о студентах по возрастанию их возрастов.

Задача 4*. Преобразовать задачи 1-3 из консольных на вывод в HTML документ с базовым (несложным) стилевым оформлением.
Рекомендуется использовать задачи предыдущих уроков.
